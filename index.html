La base

;

  • Une instruction (une action, un traitement, une phrase) finit par un ";"
  • Une instruction par ligne

Les ingrédients

Variables

  • Commencent par "$"
  • Finissent par son nom
  • Le nom commence par une minuscule
  • Le nom doit être en "camelCase"
  • Exemples : $maVariable, $titreFilm, $films
  • Une variable doit bien porter son nom !
  • "Typée"
  • Les types :
    • chaines de caractères ("string"): 'bouh', 'PHP ca déchire !', ...
    • entier ("int"), nombre décimaux ("float") : 1, 2, 3.14, ...
    • booléen ("boolean") : true, false
    • liste ("array") : array('des petits', true, 'des petits', true, 'toujours des petits', true)

Les ustensiles |

if

  • Si [condition est vraie] alors ...
    • <?php
  if (true === condition) {
  	echo 'vrai !';
  }
  • A chaque fois qu'on est entre { }, on "indente" notre code,
  • Il faut un espace entre le nom de l'ustensile et '(',
  • Entre le ')' et le '{',
  • On revient à la ligne pour '}'

foreach

  • Pour chaque clef et élément de ma liste ...

  );
  foreach ($liste as $clef => $element) {
